LucidDreamer

luciddreamer-agent — dream journal, sleep tracking, and lucid-trigger management

luciddreamer-agent is a Python package (PyPI) for recording dreams and sleep sessions, registering and attempting lucid-dream induction triggers (MILD, WBTB, SSILD, reality checks), and surfacing recurring dream signs and statistics over the log.

What This Is

luciddreamer-agent is a real Python package (PyPI). Its single public class is LucidDreamerAgent, with methods to record_dream and record_sleep, register_trigger and record_trigger_attempt for induction techniques, suggest_triggers and get_top_dream_signs for pattern analysis, and get_statistics plus export_json / import_json for summary and persistence. TriggerType covers MILD, WBTB, SSILD, RealityCheck, WakeUp, and Custom.

Quick Start

pip install "luciddreamer-agent>=0.2.2"

from luciddreamer_agent import LucidDreamerAgent, DreamMood, DreamEntry, SleepQuality, TriggerType
agent = LucidDreamerAgent()

Honesty note: luciddreamer-agent is a real, published PyPI package. Use 0.2.2 or later — 0.2.0 and 0.2.1 were published broken (a malformed __init__.py raising SyntaxError on import, and a build that accidentally bundled unrelated local files, respectively) and are both fixed as of 0.2.2. The current release installs with zero dependencies and runs standalone. The package is a small tool, not a finished app.
